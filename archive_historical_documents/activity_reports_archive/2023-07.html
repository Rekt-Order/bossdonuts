TRANSPARENCY

運営の透明化について

NFTプロジェクトには、大きな利益を生んでいるプロジェクトもあれば、個人で細々と継続しているプロジェクトもあります。私が運営する「ドーナツ部長」のプロジェクトは、今は後者よりのプロジェクトと言えるでしょう。

オンチェーンの情報だけでなく、より誠実な運営のために活動報告を行っていきます。

commit: add monthly IR reporting

企業で言うところの「IR」のようなものとお考えください。「ドーナツ部長」に関する活動、そして本サイト「ぬるぺでぃあ」に関する活動と成果を毎月公開していきます。

なぜ、活動報告を行うのか？

皆さんへ活動報告を開示する目的は、大きく2つあります。

1

活動の活性化

public function reportActivity() {
  // 報告により活動がよりアクティブになる
}
  • 毎月の活動や成果を報告することで、活動をアクティブに
  • 「しつもん箱」や「Discord」での意見を取り入れる双方向性
  • 自分自身へのプレッシャーとしても機能
2

信頼関係の構築

public function buildTrust() {
  // レポートは運営とホルダーを繋ぐ信頼の証
}
  • 運営とホルダーを繋ぐ信頼を表すアイテムとして
  • 独自の解釈と考察を含めた分析情報の提供
  • 将来的に部長ホルダーのみに限定公開する特別コンテンツへ

Conclusion & Future Commitments

2023年7月度の総括

透明性のある運営

NFTプロジェクト「ドーナツ部長」としての透明性を確保するため、月次報告を継続していきます。今後も運営とホルダーの信頼関係構築を重視します。

双方向コミュニケーション

Twitter上の参加型企画が高いエンゲージメントを獲得。8月以降は月1回の参加型コンテンツを定期開催し、コミュニティとの関係強化を図ります。

改善への取り組み

7月の反省を活かし、8月は新作NFT2点のリリース、サイトの改善（Ver.1.2）、ホワイトペーパー公開を確実に実施します。

プロジェクトの成功は、継続的な活動と信頼関係の積み重ねによって実現します。長期的なビジョンを持ちながら、着実に取り組んでいきます。
